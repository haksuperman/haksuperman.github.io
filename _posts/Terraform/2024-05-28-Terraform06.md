## 1\. terraform.tfvars의 필요성

1. 기존에는 AWS에 계정 인증을 aws configure 명령을 통해 진행했었다. **~/.aws/config** 파일과 **~/.aws/credentials** 파일을 통해 확인할 수 있었다.

2. 이 파일들이 손상된다면 AWS와의 계정 인증을 통해 코드형 인프라 작업을 할 수 없을까?

#### 1) provider 블럭에 포함시키는 방법
main.tf 파일에 직접 프로바이더 블록에 입력해주는 방법이 있다. 하지만 코드 그대로 Access key와 Secret key가 입력되기 때문에 GitHub와 같은 Repository에 올릴 수 없다.
![image](https://blog.kakaocdn.net/dna/bo0PZo/btsHGa7dvN4/AAAAAAAAAAAAAAAAAAAAALR6wzIv7ayLuHe8ToQk0nGxrwbbAGge2QDyyPdG1cTw/img.png?credential=yqXZFxpELC7KVnFOS48ylbz2pIh7yKj8&expires=1767193199&allow_ip=&allow_referer=&signature=2L91eLJqXlWG883DWT0ufT7XnsM%3D)

#### 2) variable.tf 파일에 변수를 등록시켜 입력하는 방법

1. variable.tf
    ```terraform
    variable "AWS_ACCESS_KEY" {
    default = "[access key]"
    }
    variable "AWS_SECRET_KEY" {
    default = "[secret key]"
    }
    ```
![image](https://blog.kakaocdn.net/dna/HzdV3/btsHECKDn8X/AAAAAAAAAAAAAAAAAAAAABBa9dfNeP91U7eleqyQzRHlGhjJWIzF4HM2lj_Uysfz/img.png?credential=yqXZFxpELC7KVnFOS48ylbz2pIh7yKj8&expires=1767193199&allow_ip=&allow_referer=&signature=ceV9db6kVJhKtb1Pt0DGE%2B29QD0%3D)

2. main.tf
    ```terraform
    provider "aws" {
        region = "us-west-2"
        access_key = variable.AWS_ACCESS_KEY
        secret_key = variable.AWS_SECRET_KEY
    }
    ```
![image](https://blog.kakaocdn.net/dna/vyCfx/btsHFOQSKky/AAAAAAAAAAAAAAAAAAAAAOFNyuq2My9KQd78hTVmaDtF05KxCc2cWfSoEQ0imhJS/img.png?credential=yqXZFxpELC7KVnFOS48ylbz2pIh7yKj8&expires=1767193199&allow_ip=&allow_referer=&signature=T0Mq0Rgy9awZ0NiMoaWSuaqYO2Q%3D)

## 2\. terraform.tfvars

1. variable.tf
    ```terraform
    variable "AWS_ACCESS_KEY" {}
    variable "AWS_SECRET_KEY" {}
    variable "AWS_REGION" {}
    ```
![image](https://blog.kakaocdn.net/dna/wmU46/btsHExWY5JX/AAAAAAAAAAAAAAAAAAAAAF4KsDlxHBtUQVrtP7bk2uVmRyF5nQujV6EdmmTyn6hu/img.png?credential=yqXZFxpELC7KVnFOS48ylbz2pIh7yKj8&expires=1767193199&allow_ip=&allow_referer=&signature=Y1EtULbVVA0zsh%2FNNEOrBqDwSMY%3D)

2. terraform.tfvars

    ```terraform
    AWS_ACCESS_KEY = "[access key]"
    AWS_SECRET_KEY = "[secret key]"
    AWS_REGION = "[region]"
    ```
![image](https://blog.kakaocdn.net/dna/ccWlWS/btsHF3tz3Je/AAAAAAAAAAAAAAAAAAAAAHJlHP8OaSS4w2JniQcy_ZED_jTIzrJOojo5g7Z-jlU0/img.png?credential=yqXZFxpELC7KVnFOS48ylbz2pIh7yKj8&expires=1767193199&allow_ip=&allow_referer=&signature=xac%2BeVqmKQs7LYsM34vWZpJlhSs%3D)

3. variable.tf 파일에는 변수를 정의하고, terraform.tfvars 파일에 해당 변수들의 값만 따로 저장시키는 방법이 있다.<br>
